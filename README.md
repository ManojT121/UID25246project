# 🎬 ShowBook — Movie & Event Ticketing Website

A responsive front-end ticketing platform inspired by BookMyShow, built with pure HTML, CSS, and Vanilla JavaScript. Browse movies and events, filter by category, search, and book seats — all without a backend.

---

## 🖥️ Live Demo

> Open `index.html` in any browser — no server required.

## ✨ Features

| Feature | Description |
|---|---|
| 🎠 Hero Slider | Auto-advancing banner with 3 slides, manual dot navigation |
| 🔍 Search | Live card filtering with toast feedback |
| 🗂️ Category Filter | Filter cards by Movies, Events, Music, Sports |
| 🪑 Seat Booking Modal | 24-seat grid with booked/available/selected states |
| 🔔 Toast Notifications | Lightweight popup feedback for user actions |
| 📱 Responsive Design | Mobile hamburger menu, adaptive grid layout |
| 🌙 Dark Theme | Deep dark UI with CSS custom properties |
| 🔗 Scroll-Aware Navbar | Navbar styling changes on scroll; active link tracking |

## 🚀 Getting Started

### Clone the repo

```bash
git clone https://github.com/your-username/showbook.git
cd showbook
```

### Run locally

Just open the file directly:

```bash
# macOS
open index.html

# Windows
start index.html

# Linux
xdg-open index.html
```

Or serve it with VS Code's **Live Server** extension for hot reload.
